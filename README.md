# Название проекта
## Users-система учёта пользователей и ролей.

# Описание
### Приложение предназначено для управления информацией о пользователях, ролях и операциях авторизации. Реализовано на C# (WPF) с использованием MS SQL Server и ADO.NET Entity Framework.

# Как установить
### Склонировать репозиторий в удобное вам место.

# Как запустить
Перейти в папку проекта >> папка bin >> Debug >> и запустить exe файл

# Руководство пользователя
### При запуске программы вы попадете на страницу авторизации. Вам нужно нажать на кнопку «Авторизоваться»
![Image](https://github.com/Shershakov409PS3/user/blob/project/UserWorks/Images/M5I6l1odWO.png)

### Вы попадёте на страницу авторизации, вам нужноввести свой логин и пароль. После нажать на кнопку «Войти»
![Image](https://github.com/Shershakov409PS3/user/blob/project/UserWorks/Images/BKtS9LmKqD.png)

### Вы попали на страницу изменения своих данных, в данном случае вы можете просто поменять пароль. Вводите в два поля свой новый пароль, и нажимаете сохранить. Псоле вы увидите уведомление о успешном сохранении данных.
![Image](https://github.com/Shershakov409PS3/user/blob/project/UserWorks/Images/CXZZGV4TO1.png?raw=true)
